15 Semantic Elements

This Is The Nav Element

This is the main element, it is similar to the body element but contains more central information. The main element is not seen but acts as a wrapper for viewable content.

Paragraph Element

The p tag requires opening and closing tags, and contains main content to be read by a website user.

Section Element

The section tag operates as an organizational element to define sections of a webpage. The section element utilizes opening and closing section tags and can contain Articles.

Summary & Details

This is the summary element, contained inside the details element.

The summary element is usually displayed as a header for a details element.Opening and closing tags are utilized for both summary and details elements.

Aside

This is an aside element, the aside is used to display information or content in a less than central area of the webpage. Opening and closing tags used.

Time

This is the time element, it displays dates & times. Ex: 10:00 to 21:00

Figure, Image, & Figcaption elements

The figure element contains separate content or images inside of it. The img tag is used to display images on the webpage. The figcaption element is used for accessibility to describe the image & figure content.

Mark

The mark element highlights specific text content on the webpage. In example: I will be highlighting like crazy as I study for my midterm exam.
